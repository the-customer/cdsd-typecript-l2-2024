Pour structurer il faut faire ces points:

  • Etre dans le dossier du projet
  • Initilaliser le projet par NPM : pour avoir le fichier package.json, dans lequel on a les informations du projet et aussi des paquets du projet.
    • npm init
  • Installer un serveur : lite-server
    • npm i lite-server --save-dev
  • Pour lancer le projet il faut : npm run laCleDuScript
